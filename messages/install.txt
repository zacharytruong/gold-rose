Thanks for installing Gold Rose!

Activate the color scheme:
  Preferences → Select Color Scheme… → Gold Rose

Activate the UI theme:
  Preferences → Select Theme… → Gold Rose

Both work on a stock Sublime Text 4 install with no other packages.

Optional: install "A File Icon" via Package Control for per-language file
icons in the sidebar. Gold Rose looks fine without it — you just get
Sublime's generic icons instead.
